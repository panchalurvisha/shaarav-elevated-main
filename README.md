# Shaarav Global Ventures

A premium Indian spice export company website built with React, Vite, and Tailwind CSS.

## ✨ Features

- **Modern Stack**: React 18.3+ with Vite 6 for lightning-fast development
- **Tailwind CSS**: Utility-first styling with custom design system
- **React Router**: Client-side routing for seamless navigation
- **Responsive Design**: Mobile-first approach with beautiful UI
- **Production Ready**: Optimized build with minimal dependencies
- **Clean Architecture**: Well-organized component structure

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ or Bun

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 📦 Tech Stack

### Core
- **React** 18.3.1 - UI library
- **Vite** 6.0.3 - Build tool and dev server
- **React Router** 7.1.1 - Client-side routing

### Styling
- **Tailwind CSS** 3.4.17 - Utility-first CSS framework
- **Tailwind Animate** - Animation utilities
- **Tailwind Merge** - Utility for merging Tailwind classes

### UI Components
- **Radix UI** - Accessible toast and tooltip primitives
- **Lucide React** - Beautiful icon library
- **Sonner** - Toast notifications

### State Management
- **TanStack Query** 5.62.0 - Server state management

## 🎨 Design System

The project uses a custom design system with:
- **Fonts**: Fraunces (display) and Inter (sans-serif)
- **Colors**: Warm, spice-inspired palette with terracotta, saffron, and cocoa tones
- **Components**: Editorial-style sections with grain textures and soft shadows
- **Animations**: Smooth fade-ups and marquee effects
